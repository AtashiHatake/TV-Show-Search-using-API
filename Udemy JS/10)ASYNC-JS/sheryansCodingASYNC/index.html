ASYNC JS

  • Synchronous : synchronous matlab ek ke baad dusra task hoga jab tak ek task complete nahi hoga tab tak dusra shuru nahi hoga. Eg: college lectures jab tak ek lecture khatam nahi hoga tab tak dusra shuru nahi hoga
